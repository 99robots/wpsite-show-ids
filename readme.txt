Show IDs on all post, page and taxonomy pages.

== Description ==

Features
* Adds a new row in the all posts, page or taxonomy table that displays the ID.
* Supports custom post types and custom taxonomies

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wpsite-show-ids.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
